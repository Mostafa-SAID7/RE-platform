## How to Add Screenshots

1. Take screenshots at 1920x1080 resolution for desktop views
2. Take screenshots at 375x667 resolution for mobile views
3. Use PNG format for better quality
4. Name files descriptively (e.g., `dashboard-main.png`)
5. Update this README with the new screenshot

## Screenshot Guidelines

- Use consistent browser (Chrome recommended)
- Disable browser extensions that modify UI
- Use sample data that looks realistic
- Ensure no sensitive information is visible
- Capture full page or relevant sections
- Use light mode for consistency
